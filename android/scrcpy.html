    Scrcpy Cheat Sheet¶

    Extreme quality with a more stable Video Codec¶

    scrcpy --turn-screen-off --disable-screensaver --show-touches --stay-awake --video-codec=h264 --video-bit-rate=16M --audio-bit-rate=256K --max-fps=144

    High Quality¶

    scrcpy --turn-screen-off --disable-screensaver --show-touches --stay-awake --video-codec=h265 --video-bit-rate=16M --audio-bit-rate=128K --max-fps=60

    High quality with a more stable Video Codec¶

    scrcpy --turn-screen-off --disable-screensaver --show-touches --stay-awake --video-codec=h264 --video-bit-rate=16M --audio-bit-rate=128K --max-fps=60

    Performance Optimized¶

    Lower Bitrate for Improved Performance¶

    scrcpy --max-size=1024 --video-bit-rate=2M --max-fps=30

    This command is optimized for performance, offering a good balance between quality and smoothness. The --max-size=1024 sets the maximum height or width of the video, --video-bit-rate=2M sets the video bitrate to 2 Mbps, and --max-fps=30 caps the frame rate at 30 frames per second.

    Reduced Quality for Lower Bandwidth¶

    scrcpy --max-size=800 --video-bit-rate=1M --max-fps=15

    Feature Focused¶

    For Gaming: High FPS and Show Touches¶

    scrcpy --max-fps=90 --show-touches

    Custom Configurations¶

    Recording the Screen with High Quality¶

    scrcpy --record file.mp4 --video-bit-rate=12M

    Windowed Mode with Specific Size¶

    scrcpy --window-title 'My Device' --window-width 800 --window-height 600

    Make scrcpy window always on top (above other windows).¶

    scrcpy  --always-on-top

    Encode the audio at the given bit rate, expressed in bits/s. Unit suffixes are supported: 'K' (x1000) and 'M' (x1000000).¶

    scrcpy --audio-bit-rate=value

    Select the camera size by its aspect ratio (± 10%).¶

    scrcpy --camera-ar=ar

    Specify the device camera id to mirror.¶

    scrcpy --list-cameras
scrcpy --camera-id=id

    Select the device camera by its facing direction.¶

    Possible values are "front", "back" and "external".

    scrcpy --camera-facing=facing

    Enable high-speed camera capture mode.¶

    scrcpy --camera-high-speed

    Specify an explicit camera capture size.¶

    scrcpy --camera-size=<width>x<height>

    Specify the camera capture frame rate.¶

    scrcpy --camera-fps=30

    Crop the device screen on the server.¶

    scrcpy --crop=width:height:x:y

    Use USB device (if there is exactly one, like adb -d).¶

    scrcpy --select-usb

    Disable screensaver while scrcpy is running.¶

    scrcpy --disable-screensaver

    Specify the device display id to mirror.¶

    scrcpy --display-id=0

    Add a buffering delay (in milliseconds) before displaying.¶

    scrcpy --display-buffer=0

    Use TCP/IP device (if there is exactly one, like adb -e).¶

    scrcpy --select-tcpip

    Start in fullscreen.¶

    scrcpy --fullscreen

    Do not attempt to use "adb reverse" to connect to the device.¶

    scrcpy --force-adb-forward

    This option disables these shortcuts and forwards the clicks to the device instead.¶

    scrcpy --forward-all-clicks

    Kill adb when scrcpy terminates¶

    scrcpy --kill-adb-on-close

    Simulate a physical keyboard by using HID over AOAv2.¶

    scrcpy --hid-keyboard

    Inject computer clipboard text as a sequence of key events on Ctrl+v (like MOD+Shift+v).¶

    scrcpy --legacy-paste

    List device cameras.¶

    scrcpy --list-cameras

    List the valid camera capture sizes.¶

    scrcpy --list-camera-sizes

    List device displays.¶

    scrcpy --list-displays

    List video and audio encoders available on the device.¶

    scrcpy --list-encoders

    Lock video orientation to value.¶

    scrcpy --lock-video-orientation[=value]

    Limit both the width and height of the video to value.¶

    scrcpy --max-size=value

    Simulate a physical mouse by using HID over AOAv2.¶

    scrcpy --hid-mouse

    Limit the frame rate of screen capture (officially supported since Android 10, but may work on earlier versions).¶

    scrcpy --max-fps=value

    Disable device control (mirror the device in read-only)¶

    scrcpy  --no-control

    Disable video and audio playback on the computer (equivalent to --no-video-playback --no-audio-playback).¶

    scrcpy --no-playback

    Disable audio forwarding.¶

    scrcpy --no-audio

    Disable audio playback on the computer.¶

    scrcpy --no-audio-playback

    This option disables this cleanup.¶

    scrcpy --no-cleanup

    This option disables this automatic synchronization.¶

    scrcpy --no-clipboard-autosync

    By default, on MediaCodec error, scrcpy automatically tries again with a lower definition.¶

    scrcpy --no-downsize-on-error

    Do not forward repeated key events when a key is held down.¶

    scrcpy --no-key-repeat

    If the renderer is OpenGL 3.0+ or OpenGL ES 2.0+,¶

    scrcpy --no-mipmaps

    Do not power on the device on start.¶

    scrcpy --no-power-on

    Disable video forwarding.¶

    scrcpy --no-video

    Disable video playback on the computer.¶

    scrcpy --no-video-playback

    et the TCP port (range) used by the client to listen.¶

    scrcpy --port=27183:27199

    Configure pause on exit.¶

    scrcpy --pause-on-exit[=mode]

    Turn the device screen off when closing scrcpy.¶

    scrcpy --power-off-on-close

    Inject alpha characters and space as text events instead of key events.¶

    scrcpy --prefer-text

    Start FPS counter, to print framerate logs to the console. It can be started or stopped at any time with MOD+i.¶

    scrcpy --print-fps

    Set the target directory for pushing files to the device by drag & drop.¶

    scrcpy --push-target=path

    Record screen to file.¶

    scrcpy --record=file.mp4

    Inject key events for all input keys, and ignore text events.¶

    scrcpy --raw-key-events

    Force recording format (either mp4 or mkv).¶

    scrcpy --record-format=mkv

    Request SDL to use the given render driver (this is just a hint).¶

    Supported names are currently "direct3d", "opengl", "opengles2", "opengles", "metal" and "software".

    scrcpy --render-driver=name

    This option makes scrcpy fail if audio is enabled but does not work.¶

    scrcpy --require-audio

    Set the initial display rotation.¶

    scrcpy --rotation=value

    The device serial number. Mandatory only if several devices are connected to adb.¶

    scrcpy  --serial=serial

    Turn the device screen off immediately.¶

    ```bash scrcpy --turn-screen-off

    Set the maximum mirroring time, in seconds.¶

    scrcpy --time-limit=seconds

    scrcpy --tunnel-host=ip

    scrcpy --tunnel-port=port

    Set the log level (verbose, debug, info, warn or error).¶

    scrcpy --verbosity=info

    scrcpy --v4l2-sink=/dev/videoN

    Add a buffering delay (in milliseconds) before pushing frames. This increases latency to compensate for jitter.¶

    scrcpy --v4l2-buffer=ms

    Select a video codec (h264, h265 or av1).¶

    scrcpy --video-codec=name

    Set a list of comma-separated key:type=value options for the device video encoder.¶

    scrcpy --video-codec-options=key[:type]=value[,...]

    Use a specific MediaCodec video encoder (depending on the codec provided by --video-codec).¶

    scrcpy --video-encoder=name

    Select the video source (display or camera).¶

    scrcpy --video-source=source

    Keep the device on while scrcpy is running, when the device is plugged in.¶

    scrcpy  --stay-awake

    Disable window decorations (display borderless window).¶

    scrcpy --window-borderless

    Set a custom window title.¶

    scrcpy --window-title=text

    Set the initial window horizontal position.¶

    scrcpy --window-x=value

    Set the initial window vertical position.¶

    scrcpy --window-y=value

    Set the initial window width.¶

    scrcpy --window-width=value

    Set the initial window height.¶

    scrcpy --window-height=value
